## ⚕️복잡한 일반의약품 정보, 이제 다국어로 쉽게 이해할 수 있어요!<br>
### 🥈경기대학교 AI컴퓨터공학부 기초캡스톤 경진대회 은상 수상작
K-Mediguide 는 식약처에서 제공하는 e약은요 api 를 활용하여 이에 기반한 정확한 일반의약품 정보를 추출합니다.<br> 또한 RAG 기술을 적용하여, 공식문서에 기반한 답변을 제공할 수 있도록 구현하였습니다.
사용자의 질문으로부터 유사한 답변을 사용하여 context 를 추출하고 이를 OpenAI 의 GPT3.5 모델에게 제공하여 이에 기반한 정확한 답변을 생성하도록 구현하였습니다.
